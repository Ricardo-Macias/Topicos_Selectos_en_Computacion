# Practica 13: Real Time Clock (RTC) con LCD

En esta practica se hara uso de un circuito integrado de bajo consumo que mantiene la hora actual.

## Materiales

- Arduino.
- LCD.
- Potenciometro.
- RTC.

## Librerias

- RTClib
- LiquidCrystal

## Conexiones

| **RTC** | **Arduino** |
|:-------:|:-----------:|
| 5V      | 5V          |
| GND     | GND         |
| SCL     | A5          |
| SDA     | A4          |
