# Practica 12: Sensor ultrasónico HC-SR04 y LCD

En esta practica se creara un programa capaz de medir que tan lejos se encunetra un objeto, mostrar la distancia en el LCD.

## Materiales

- Arduino.
- LCD.
- Potenciometro.
- Sensor ultrasonido HC-SR04.

## Librerias

- LiquidCrystal
